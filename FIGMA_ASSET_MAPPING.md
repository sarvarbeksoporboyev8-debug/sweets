# Figma Asset Mapping

This file maps Figma MCP asset IDs to local asset filenames.

## How to Export Images from Figma

1. Open your Figma file
2. For each asset ID below, find the corresponding image in Figma
3. Export the image as PNG
4. Save it to `assets/images/figma/` with the filename: `{asset-id}.png`
